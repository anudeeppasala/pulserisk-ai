# PulseRisk AI Demo Script

## One-Line Pitch

PulseRisk AI converts customer feedback into risk signals, issue clusters, root-cause suggestions, action tickets, and executive dashboards.

## Demo Flow

1. Open the Streamlit dashboard.
2. Explain that the system accepts app reviews, support tickets, survey comments, and customer feedback.
3. Show top metrics:
   - Total comments
   - Negative comments
   - High / critical issues
   - Issue clusters
   - Action tickets
4. Show the executive summary.
5. Show issue category chart.
6. Show severity breakdown.
7. Show risk type breakdown.
8. Show owner team routing.
9. Open the Issue Clusters tab.
10. Show root-cause suggestions.
11. Open the Action Tickets tab.
12. Show Jira-style ticket generation.
13. Open Trends & Version Impact.
14. Show daily comment volume, spike signals, and version impact.
15. Download classified results or tickets.

## Key Talking Points

- V1 classified individual comments.
- V2 groups comments into issue clusters.
- V2 suggests possible root causes.
- V2 generates action tickets from recurring issues.
- The platform turns unstructured text into structured risk and product intelligence.
- The classifier works without API keys, but the architecture is ready for LLM classification.
